## 프로젝트 소개 및 기능
<p>안경 브랜드인 FAKEME의 홈페이지를 새롭게 디자인하고 React를 활용하여 구현한 프로젝트</p>
<ul>
  <li>메인 페이지, 그리고 각각의 메뉴 클릭 시 페이지 이동</li>
  <li>상품 메뉴 클릭 시 해당되는 상품들 나열</li>
  <li>언어 변경 시 useTranslate를 활용하여 맞는 언어가 렌더링</li>
</ul>
